```bash

```
Відкриє `http://localhost:5173` автоматично

### Збірка проєкту
```bash
npm run build
```
Створить папку `dist/` з готовим продакшн кодом

### Додавання нових модулів
1. Створіть файл в `src/script/yourModule.ts`
2. Додайте в `main.ts`:
   ```typescript
   import './yourModule';
   ```

## 🎯 Функціональність

### Counter (Счетчик)
- **Файл:** `src/script/counter.ts`
- **Функції:**
  - `+1` - збільшує на 1
  - `-1` - зменшує на 1
  - `+2` - збільшує на 2
  - `-2` - зменшує на 2
  - Автоматичний wrap: при >=100 скидається на (value-100), при <=-100 на (value+100)

### Модульна архітектура
- `main.ts` - точка входу, імпортує всі модулі
- `counter.ts` - окрема логіка счетчика
- `new_step.ts` - консоль лог "Hello World!"
